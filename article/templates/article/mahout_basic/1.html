技术背景

在接触Mahout之前一直是做网站的Web开发，因此主要是对HTML/CSS/Javascript/JQuery/PHP/MySQL这些方面比较精通, 没有用Java进行过项目的开发所以走过一些弯路, 因此在下面我总结了一些Mahout入门需要的知识点。

有助于Mahout入门的一些知识点

  • Mahout是一个maven的项目, 因此你需要对于maven有一些基本的了解。下面我只列出Mahout二次开发需要掌握的一些内容

    1. 下载Mahout的源码包, 由于我们要进行二次开发，所以请下载mahout-distribution-0.8-src.tar.gz
    2. 解压后cd到解压目录，然后执行命令 mvn install -Dmaven.test.skip=true, 如果你是第一次接触maven(我接触mahout的情况就是如此), 请一定记得加上后面的-Dmaven.skip.test=true 要知道如果不加上将会运行mahout里所有的UnitTest，那安装时间就海了去了
    3. 安装过程中将会自动下载依赖的jar包，因此需要等待几分钟来完成
    4. 以后我们进行Mahout的二次开发修改代码后都要执行mvn install -Dmaven.test.skip=true来确保代码被重新编译和打包

    安装完成后的操作

    • Mahout安装完后需要一些设置，下面是我自己电脑里面的/etc/profile添加的内容

      /etc/profile
      export JAVA_HOME=/usr/lib/jvm/java-1.6.0-openjdk-1.6.0.0
export HADOOP_HOME=/home/hadoop-user/hadoop-1.2.1
export HADOOP_CONF_DIR=$HADOOP_HOME/conf
export MAHOUT_HOME=/home/hadoop-user/workspace/mahout-distribution-0.8
export M2_HOME=/home/hadoop-user/apache-maven-3.1.0
export M2=$M2_HOME/bin
export MONGODB_HOME=/home/hadoop-user/mongodb
export COBERTURA_HOME=/home/hadoop-user/cobertura-2.0.3
export PYTHON_HOME=/usr/local/python2.7
export GAE_HOME=/usr/local/google_appengine
export TOPKNOWBLOG=/home/hadoop-user/workspace/topknowblog
export PATH=$JAVA_HOME/bin:$MAHOUT_HOME/bin:$M2:$HADOOP_HOME/bin:$COBERTURA_HOME:$MONGODB_HOME/bin:$GAE_HOME:$PYTHON_HOME/bin:$PATH
export CLASSPATH=$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$MAHOUT_HOME/lib:HADOOP_HOME/lib
export PYTHONPATH=$TOPKNOWBLOG:$TOPKNOWBLOG/autoload:$TOPKNOWBLOG/dbindexer:$TOPKNOWBLOG/django:$TOPKNOWBLOG/djangoappengine:$TOPKNOWBLOG/djangotoolbox
