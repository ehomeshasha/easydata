Mahout安装流程

  1. 安装maven

  2. 安装Hadoop并配置好环境变量, 下面是我自己在/etc/profile添加的一些环境变量设置

    {% endblocktrans %} {% get_code 1 %} {% blocktrans %}
  3. 下载Mahout的源码包, 由于我们要进行二次开发，所以请下载mahout-distribution-0.8-src.tar.gz
  4. 解压后cd到解压目录，然后执行命令 mvn install -Dmaven.test.skip=true, 如果你是第一次接触maven(我接触mahout的情况就是如此), 请一定记得加上后面的-Dmaven.skip.test=true 要知道如果不加上将会运行mahout里所有的UnitTest，会非常耗费时间。
  5. 安装过程中将会自动下载依赖的jar包，因此需要等待几分钟来完成
  6. 以后我们进行Mahout的二次开发修改代码后都要执行mvn install -Dmaven.test.skip=true来确保代码被重新编译和打包

  7. 将Mahout导入Eclipse(须先安装maven插件m2eclipse), 这里我附上一张我导入项目后的截图
