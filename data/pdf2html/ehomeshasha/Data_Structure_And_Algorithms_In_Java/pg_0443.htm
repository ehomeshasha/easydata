Adding Vertices and Edges to a Graph
To add a vertex to a graph, you make a new vertex object with
new
and insert it into your
vertex array,
vertexList
. In a real-world program a vertex might contain many data

creation of a vertex looks something like this:
vertexList[nVerts++] = new Vertex('F');
This inserts a vertex F, where
nVerts
is the number of vertices currently in the graph.
