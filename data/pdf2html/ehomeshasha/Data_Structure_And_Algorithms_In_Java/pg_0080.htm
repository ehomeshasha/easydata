Java Code for Insertion Sort
Here's the method that carries out the insertion sort, extracted from the
insertSort.java
program:
public void insertionSort()
{
int in, out;
for(out=1; out<nElems; out++) // out is dividing line
{
double temp = a[out]; // remove marked item
in = out; // start shifts at out
while(in>0 && a[in-1] >= temp) // until one is smaller,
{
a[in] = a[in-1]; // shift item right,
--in; // go left one position
}
a[in] = temp; // insert marked item
} // end for
} // end insertionSort()
In the outer
for
loop,
out
starts at 1 and moves right. It marks the leftmost unsorted
data. In the inner
while
loop,
in
starts at
out
and moves left, until either
temp
is
smaller than the array element there, or it can't go left any further. Each pass through the
while
loop shifts another sorted element one space right.
